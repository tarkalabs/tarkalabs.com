This is the second article that I am writing in the efforts to create a neural network that will colorize a grayscale image. In the previous article I explored the shape and structure of an image present in the miniplaces dataset. In this article, I will be preparing the dataset for training and testing. Also I will be writing a getbatch function that will come in handy for training our network.

I am going to split this article into 3 sections.

  • Preparing the training dataset.
  • Preparing the testing dataset.
  • Writing the getbatch function.

The code used in this article can be in this github repo https://github.com/deepakprasanna/tfcolorize in the notebook 01_dataprep.

Preparing the training dataset

Before we get started please keep in mind that since we are building a neural-net to colorize grayscale images, our X will be the pixel values of all grayscale images and Y will be the pixel values of all colorized images.

import numpy as np
import cv2
import matplotlib.pyplot as plt
from glob import glob
%matplotlib inlinetrain_files = glob("./data/images/train/**/*.jpg", recursive=True)
train_files.sort()train = {}
train["labels"] = np.array([f.split("/")[5] for f in train_files])
train["X"] = np.array([cv2.imread(f, cv2.IMREAD_COLOR) for f in train_files])
train["Y"] = np.array([cv2.imread(f, cv2.IMREAD_GRAYSCALE) for f in train_files])

First thing that I am doing in the above code is getting the path to all the files in CIFAR-10 dataset. My idea is to store the data for training in a dictionary called train and test for test respectively. Then I proceed to store the category to which the image belongs in train["labels"], RGB pixel values in train["X"] and grayscale pixel values in train["Y"].

Now, lets print the shapes of train["labels"], train["X"] and train["Y"].

Let’s reshape train["X"] and train["Y"].

train["X"] = np.reshape(train["X"], [100000, 1, 128, 128])
train["Y"] = np.swapaxes(np.swapaxes(train["Y"], 2, 3), 1, 2)
print(train["X"].shape)
print(train["Y"].shape)

Now we can see that train["X"] and train["Y"] got reshaped as we wanted. It is the same reshaping we did in the previous part but we are doing it for the entire dataset this time.

Also lets print all the unique categories which are stored in train["labels"].

As per the details mentioned in the miniplaces dataset, it contains images from 100 categories. Since the shape of unique labels is (100,), we are good.

Preparing the testing dataset

The steps for preparing the test dataset is very similar to what we did in train.

test_files = glob("./data/images/test/*.jpg", recursive=True)
test_files.sort()test = {}
test["X"] = np.array([cv2.imread(f, cv2.IMREAD_GRAYSCALE) for f in test_files])
test["Y"] = np.array([cv2.imread(f, cv2.IMREAD_COLOR) for f in test_files])test["X"] = np.reshape(test["X"], [10000, 1, 128, 128])
test["Y"] = np.swapaxes(np.swapaxes(test["Y"], 2, 3), 1, 2)

The main difference between test and train dataset is that we don’t have the label information and the number of samples for test is 10000.

Writing the getbatch function

Often when you train your model in deeplearning, you need a getbatch function which will give you random batch of samples from the dataset. Let’s write agetbatch function which will give random batches of X and Y from our dataset.

def getbatch(X, Y, nsamples, batchsize = 1000):
    randstart = np.random.randint(0, nsamples - batchsize - 1)
    count = 0
    while count < nsamples/batchsize:
        start = count*batchsize
        count += 1
        yield (X[randstart:randstart+batchsize], Y[randstart:randstart+batchsize])

Python supports a feature called generator functions, the above getbatch function is a generator function. If you are not familiar with the yield keyword, I highly recommend reading up what generator functions are first. Otherwise this code is straight forward, if you invoke getbatch with train["X"] and train["Y"] , you will get random batches of X and Y with a batchsize of 1000. This will be useful when we are going to train our model.

Let’s run and test our getbatch function.

Perfect, the getbatch function works as expected. In the next part I will create a model and start the training process.
