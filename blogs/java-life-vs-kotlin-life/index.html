Kotlin provides many amazing and powerful features over Java. Developers always love clean and concise code. Less code takes less time to write and less time to read and makes it easy to maintain.

Kotlin has concise code compared to Java. It drastically reduces the amount of boilerplate code, increasing your development’s speed. Kotlin simplifies the developer’s life with its powerful features. We’ll see how in this blog by diving into Kotlin’s features.

Why this blog?

Before Kotlin was introduced, Java was the only language to develop Native android applications. Nowadays, most developers are loving & using Kotlin. But, many of them use Kotlin, like how they’ve used Java. They’re still living Java life, even while using Kotlin. This blog post is to help bring them into Kotlin Life and use it to its fullest potential.

Kotlin features over Java

1. Filtering a list and returning another list:

Let’s say we have a List<Developer> and we need to get all the Developers whose platform is Android.

Using Java:

public class DeveloperExample {
  // Using Default way
  private List<Developer> getAndroidDevelopersByUsualWay(List<Developer> developers) {
    List<Developer> androidDevelopers = new ArrayList<>();

    for (int i = 0, size = developers.size(); i < size; i++) {
      Developer developer = developers.get(i);
      if (developer.getPlatform() == Developer.Platform.ANDROID) {
        androidDevelopers.add(developer);
      }
    }
    return androidDevelopers;
  }

  // Using ForEach
  private List<Developer> getAndroidDevelopersByForEach(List<Developer> developers) {
    List<Developer> androidDevelopers = new ArrayList<>();

    developers.forEach(developer -> {
      if (developer.getPlatform() == Developer.Platform.ANDROID) {
        androidDevelopers.add(developer);
      }
    });
    return androidDevelopers;
  }

  // Using Java Streams
  private List<Developer> getAndroidDevelopersByJavaStreams(List<Developer> developers) {
    return developers.stream()
        .filter(developer -> developer.getPlatform() == Developer.Platform.ANDROID)
        .collect(Collectors.toList());
  }
}

Using Kotlin (filter Operator):

class KotlinFilterExample {
  fun getAndroidDevelopers(developers: List<Developer>) : List<Developer> {
   return developers.filter{ developer ->developer.platform == ANDROID }
  }
}

2. Converting a list into another:

Let’s say we have a List<Developer> and we need to get all the Developer's names alone.

Using Java:

public class JavaConvertingListToAnotherExample {
  // Using Default way
  private List<String> getDeveloperNamesByUsualWay(List<Developer> developers) {
    List<String> developerNames = new ArrayList<>();

    for (int i = 0, size = developers.size(); i < size; i++) {
      Developer developer = developers.get(i);
      developerNames.add(developer.getName());
    }
    return developerNames;
  }

  // Using ForEach
  private List<String> getDeveloperNamesByForEach(List<Developer> developers) {
    List<String> developerNames = new ArrayList<>();

    developers.forEach(developer -> {
      developerNames.add(developer.getName());
    });
    return developerNames;
  }

  // Using Java Streams
  private List<String> getDeveloperNamesByStreams(List<Developer> developers) {
    return developers.stream()
        .map(Developer::getName)
        .collect(Collectors.toList());
  }
}

Using Kotlin (map & mapNotNull Operator):

class KotlinConvertingAListExample {
  // Using mapNotNull
  fun getDeveloperNamesByMap(developers: List<Developer>): List<String> {
    return developers.map { developer -> developer.name }
  }

  // Using mapNotNull
  fun getDeveloperNamesByMapNotNull(developers: List<Developer>): List<String> {
    return developers.mapNotNull { developer -> developer.name }
  }
}

3. Find an element in a list:

Let’s say we have a List<Developer> and we need to find a Developer whose name is John. Query = “John”.

Using Java:

public class JavaFindAnElementFromList {
  // Using Default Way
  private Developer getDeveloperByDefaultWay(List<Developer> developers, String query) {
    Developer developer = null;
    for (int i = 0, size = developers.size(); i < size; i++) {
      Developer developer1 = developers.get(i);
      if (developer1.getName().equalsIgnoreCase(query)) {
        developer = developer1;
        break;
      }
    }
    return developer;
  }

  // Using Streams
  private Developer getDeveloperByStreams(List<Developer> developers, String query) {
    return developers.stream()
        .filter(developer -> developer.getName().equalsIgnoreCase(query))
        .findAny()
        .orElse(null);
  }
}

Using Kotlin: (find Operator)

class KotlinFindAnElementExample {
  // Using Find
  fun getDeveloper(developers: List<Developer>, query: String): Developer? {
    return developers.find { developer -> developer.name.contains(query) }
  }
}
